Benchmark round: empty-contract-evaluate

Test an evaluateTransaction() Gateway method against the NodeJS `fixed-asset` Smart Contract method named `nullResponse`, which immediately returns a null response. This represents the fastest possible round trip time for an evaluateTransaction() method that does not touch the world state or perform any action.

txDuration: 300
rateControl:
  type: fixed-load
  opts:
    transactionLoad: 2000
